## Bugs found & fixed this run

1. **Router:** `initFromUrl()` did not reset `sessionStorage` when URL was `landing` — stale step stuck on worries. **Fixed** in `funnel-state.js` (always `setStep` from URL).
2. **IAB height:** Shell used `100dvh` only — risk on Meta in-app browser first paint. **Fixed** → `100svh` + fallbacks.
3. **Safe area:** Footer relied on `env(safe-area-inset-bottom)` alone — often `0` in IAB. **Fixed** → `max(16px, env(...))`.
